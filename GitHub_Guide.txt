Push 전엔 Pull이 있다!

------------------------------------------------------

pull

git pull origin master
- pull 하는 커멘드, 지정 해놓은 repository에서 끌어옵니다.

git status
- 현재 Git 과의 상태를 체크

git add .
- 변경된 파일들을 모두 Tracking

git commit -m "message"
- 인식 할 수 있는 commit을 달아줍니다.

------------------------------------------------------

push

git push origin master
- 실제로 Git에 Push!
